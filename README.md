# CTKCHPSDK (iOS SPM)

Swift Package Manager (SPM) release repository for the **CTKCHPSDK** iOS library.

---

## Installation

To add **CTKCHPSDK** to your Xcode project:

1. Open Xcode and select **File > Add Packages...**.
2. Enter the repository URL in the search bar:
   ```text
   https://github.com/michaelleechoicetech/CTKCHPSDK-iOS.git
   ```
3. Set the Dependency Rule to **Up to Next Major Version** (e.g. `1.0.0`) or select the branch name.
4. Click **Add Package**.
